# 日志监控器 (Log Monitor)

一个基于 Python tkinter 的实时日志、JSON 与 CSV 数据监控工具，支持自由拖拽面板、可配置刷新频率。

## 环境要求

- Python 3.6+
- 标准库（tkinter, json, csv, os），无需额外安装

## 快速开始

```bash
python log_monitor.py
```

也可指定自定义配置文件：

```bash
python log_monitor.py my_config.json
```

## 模块结构

| 文件 | 说明 |
|------|------|
| `log_monitor.py` | 主模块，应用入口、工具栏、配置管理、对话框 |
| `log_panel.py` | 日志监控面板模块 |
| `json_panel.py` | JSON 监控面板模块 |
| `table_panel.py` | CSV 表格监控面板模块 |
| `config.json` | 配置文件 |

## 配置文件 (`config.json`)

```json
{
    "logs": [
        {
            "path": "example1.log",
            "lines": 30,
            "refresh_ms": 1000,
            "order": "asc"
        }
    ],
    "json_monitors": [
        {
            "path": "example.json",
            "fields": ["code", "data.host.hostname"],
            "field_aliases": {"code": "状态码"},
            "refresh_ms": 1000
        }
    ],
    "table_monitors": [
        {
            "path": "example.csv",
            "columns": ["name", "score"],
            "column_aliases": {"name": "姓名", "score": "分数"},
            "max_rows": 100,
            "refresh_ms": 1000
        }
    ]
}
```

### 日志监控项 (`logs`)

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `path` | string | — | 日志文件路径（相对或绝对） |
| `lines` | int | — | 显示最后多少行 |
| `refresh_ms` | int | — | 刷新间隔（毫秒） |
| `order` | string | `"asc"` | 排序方向：`"asc"`（旧→新）/ `"desc"`（新→旧） |

### JSON 监控项 (`json_monitors`)

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `path` | string | — | JSON 文件路径 |
| `fields` | list | — | 要监控的字段列表（最多 10 个），用点号分隔层级，如 `"data.host.hostname"` |
| `field_aliases` | object | `{}` | 字段别名，格式 `{"字段名": "别名"}`，显示时优先使用别名 |
| `refresh_ms` | int | — | 刷新间隔（毫秒） |

### CSV 表格监控项 (`table_monitors`)

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `path` | string | — | CSV 文件路径 |
| `columns` | list | — | 要显示的列名列表 |
| `column_aliases` | object | `{}` | 列别名，格式 `{"列名": "别名"}`，显示时优先使用别名 |
| `max_rows` | int | — | 最大显示行数 |
| `refresh_ms` | int | — | 刷新间隔（毫秒） |

## 界面操作

### 工具栏

| 按钮 | 功能 |
|------|------|
| **添加日志** | 选择日志文件 → 设置行数/刷新间隔/排序 → 添加面板 |
| **删除日志** | 列表中选择已监控的日志文件 → 删除 |
| **添加JSON** | 选择 JSON 文件 → 自动解析字段 → 多选最多 10 个字段 → 双击可设置别名 → 设置刷新间隔 |
| **编辑JSON** | 选择已监控的 JSON → 重新选择字段和别名 |
| **删除JSON** | 列表中选择已监控的 JSON → 删除 |
| **添加表格** | 选择 CSV 文件 → 自动解析表头 → 多选要显示的列 → 双击可设置别名 → 设置行数/刷新间隔 |
| **编辑表格** | 选择已监控的 CSV → 重新选择列和别名 |
| **删除表格** | 列表中选择已监控的 CSV → 删除 |
| **重新载入** | 重新加载配置文件 |

### 面板操作

- **拖拽**：按住面板标题栏拖动，可自由调整面板位置
- **缩放**：拖动面板右下角的 ◢ 图标，可调整面板大小
- **弹出**：点击「弹出」按钮，面板脱离主窗口成为独立窗口
- **收回**：关闭弹出的独立窗口，面板回到主窗口
- **位置持久化**：面板位置和大小自动保存到配置文件

### 界面布局

```
┌──────────────────────────────────────────────────────┐
│  添加日志 删除日志 │ 添加JSON 编辑JSON 删除JSON │ ... │
├──────────────────────────────────────────────────────┤
│  ┌──────────────┐  ┌──────────────┐                  │
│  │ example1.log │  │ example.json │                  │
│  │              │  │ code: 0      │                  │
│  │              │  │ hostname: .. │   ┌───────────┐  │
│  │              │  │              │   │ test.csv  │  │
│  └──────────────┘  └──────────────┘   │ 姓名 分数 │  │
│          ◢               ◢            │ 张三  95  │  │
│                                       └───────────┘  │
│                                              ◢       │
└──────────────────────────────────────────────────────┘
```

- 三种面板（日志/JSON/表格）在同一个主窗口中自由排列，不再按列固定
- 暗色主题（黑底浅色字）显示日志和 JSON 面板
- 表格面板使用 Treeview 组件，带水平和垂直滚动条
