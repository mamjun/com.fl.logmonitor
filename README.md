# 日志监控器 (Log Monitor)

一个基于 Python tkinter 的实时日志、JSON 与 CSV 数据监控工具，采用现代暗色主题，支持自由拖拽面板、面板弹出/收回、双击重命名、可配置刷新频率，以及面板位置持久化。

## 环境要求

- Python 3.6+
- 标准库（tkinter, json, csv, os），无需额外安装

## 快速开始

```bash
python log_monitor.py
```

也可指定自定义配置文件：

```bash
python log_monitor.py my_config.json
```

### 打包为 exe

项目包含 PyInstaller 配置文件 `log_monitor.spec`，可直接打包：

```bash
pip install pyinstaller
pyinstaller log_monitor.spec
```

打包后的 exe 位于 `dist/` 目录下。

## 模块结构

| 文件 | 说明 |
|------|------|
| `log_monitor.py` | 主模块，应用入口、暗色主题、工具栏、状态栏、配置管理、对话框 |
| `log_panel.py` | 日志监控面板模块 |
| `json_panel.py` | JSON 监控面板模块 |
| `table_panel.py` | CSV 表格监控面板模块 |
| `config.json` | 配置文件 |
| `log_monitor.spec` | PyInstaller 打包配置 |

## 配置文件 (`config.json`)

### 全局配置

| 字段 | 类型 | 说明 |
|------|------|------|
| `window_width` | int | 主窗口宽度（自动保存） |
| `window_height` | int | 主窗口高度（自动保存） |

### 面板位置配置（自动管理）

每个监控项可包含以下位置字段，由程序自动保存和恢复：

| 字段 | 类型 | 说明 |
|------|------|------|
| `x` | int | 面板在内容区域内的 X 坐标 |
| `y` | int | 面板在内容区域内的 Y 坐标 |
| `width` | int | 面板宽度 |
| `height` | int | 面板高度 |
| `detached` | bool | 面板是否处于弹出状态 |
| `geometry` | string | 弹出窗口的几何信息（如 `"500x350+100+100"`） |
| `display_name` | string | 面板自定义显示名称（双击标题可编辑） |

### 配置示例

```json
{
    "window_width": 1200,
    "window_height": 800,
    "logs": [
        {
            "path": "example1.log",
            "lines": 30,
            "refresh_ms": 1000,
            "order": "asc",
            "display_name": "我的应用日志",
            "x": 10,
            "y": 10,
            "width": 400,
            "height": 300
        }
    ],
    "json_monitors": [
        {
            "path": "example.json",
            "fields": ["code", "data.host.hostname"],
            "field_aliases": {"code": "状态码"},
            "refresh_ms": 1000,
            "display_name": "服务器状态",
            "x": 420,
            "y": 10,
            "width": 400,
            "height": 300
        }
    ],
    "table_monitors": [
        {
            "path": "example.csv",
            "columns": ["name", "score"],
            "column_aliases": {"name": "姓名", "score": "分数"},
            "max_rows": 100,
            "refresh_ms": 1000,
            "display_name": "成绩表",
            "x": 830,
            "y": 10,
            "width": 400,
            "height": 300
        }
    ]
}
```

### 日志监控项 (`logs`)

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `path` | string | — | 日志文件路径（相对或绝对） |
| `lines` | int | — | 显示最后多少行 |
| `refresh_ms` | int | — | 刷新间隔（毫秒） |
| `order` | string | `"asc"` | 排序方向：`"asc"`（旧→新）/ `"desc"`（新→旧） |
| `display_name` | string | 文件名 | 面板自定义显示名称，不设置则显示文件名 |

### JSON 监控项 (`json_monitors`)

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `path` | string | — | JSON 文件路径 |
| `fields` | list | — | 要监控的字段列表（最多 10 个），用点号分隔层级，如 `"data.host.hostname"` |
| `field_aliases` | object | `{}` | 字段别名，格式 `{"字段名": "别名"}`，显示时优先使用别名 |
| `refresh_ms` | int | — | 刷新间隔（毫秒） |
| `display_name` | string | 文件名 | 面板自定义显示名称，不设置则显示文件名 |

### CSV 表格监控项 (`table_monitors`)

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `path` | string | — | CSV 文件路径 |
| `columns` | list | — | 要显示的列名列表 |
| `column_aliases` | object | `{}` | 列别名，格式 `{"列名": "别名"}`，显示时优先使用别名 |
| `max_rows` | int | — | 最大显示行数 |
| `refresh_ms` | int | — | 刷新间隔（毫秒） |
| `display_name` | string | 文件名 | 面板自定义显示名称，不设置则显示文件名 |

## 界面操作

### 工具栏

| 按钮 | 功能 |
|------|------|
| **📄 添加日志** | 选择日志文件 → 设置行数/刷新间隔/排序 → 添加面板 |
| **🗑 删除日志** | 列表中选择已监控的日志文件 → 删除 |
| **📋 添加JSON** | 选择 JSON 文件 → 自动解析字段 → 多选最多 10 个字段 → 双击可设置别名 → 设置刷新间隔 |
| **✏ 编辑JSON** | 选择已监控的 JSON → 重新选择字段和别名 |
| **🗑 删除JSON** | 列表中选择已监控的 JSON → 删除 |
| **📊 添加表格** | 选择 CSV 文件 → 自动解析表头 → 多选要显示的列 → 双击可设置别名 → 设置行数/刷新间隔 |
| **✏ 编辑表格** | 选择已监控的 CSV → 重新选择列和别名 |
| **🗑 删除表格** | 列表中选择已监控的 CSV → 删除 |
| **💾 保存位置** | 将当前所有面板位置、大小及弹出窗口状态保存到配置文件 |
| **🔄 重置位置** | 清除所有面板位置信息，下次加载时使用默认布局 |
| **↻ 重新载入** | 重新加载配置文件，刷新所有面板 |

### 面板操作

- **拖拽**：按住面板标题栏拖动，可自由调整面板位置
- **缩放**：拖动面板右下角的 ◢ 图标，可调整面板大小
- **弹出**：点击标题栏右侧的 ⬈ 按钮，面板脱离主窗口成为独立窗口
- **收回**：关闭弹出的独立窗口，面板回到主窗口
- **重命名**：双击面板标题，可编辑自定义显示名称，按 Enter 保存，按 Esc 取消
- **位置持久化**：面板位置和大小自动保存到配置文件

- 三种面板（日志/JSON/表格）在同一个主窗口中自由拖拽排列，位置和大小自动保存到配置文件

- 底部状态栏实时显示当前监控项数量统计
